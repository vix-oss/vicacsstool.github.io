# Bloxd.io CSS 紋理包生成器

## 介紹

**Bloxd CSS 生成器** 是一個強大的線上工具，幫助 Bloxd.io 玩家輕鬆自訂遊戲的視覺效果。無需編寫複雜的 CSS 代碼，只需選擇功能模組並調整參數，即可生成專屬的紋理包樣式。

## 功能特色

### 🎯 支援超過 25+ 個自訂模組

- **準心自訂** - 上傳自己的準心圖片
- **廣告隱藏** - 隱藏遊戲內廣告
- **聊天美化** - 自訂聊天室背景、邊框、陰影效果
- **快捷欄與物品** - 調整熱鍵欄和背包樣式
- **血條樣式** - 自訂生命值顯示
- **主選單背景** - 個性化主菜單
- **手機版按鈕** - 手機友善的按鈕設計
- **多種主題** - 包括設定選單、按鈕、社交欄、輸入框、背包、商店、角色、載入器等主題

### ✨ 核心優勢

✅ **無需代碼知識** - 直觀的圖形界面，任何人都能使用  
✅ **實時預覽** - 即時查看生成的 CSS 代碼  
✅ **一鍵複製** - 輕鬆複製生成的 CSS 代碼到剪貼板  
✅ **模組化設計** - 自由組合不同功能模組  
✅ **繁體中文界面** - 完整的繁體中文支援  

## 使用方法

1. 打開 [texture.html](texture.html)
2. 從下拉菜單選擇想要自訂的功能模組
3. 點擊「+ 新增功能」按鈕
4. 調整各項參數（顏色、大小、位置等）
5. 在右側「CSS 代碼預覽」區域查看生成的代碼
6. 點擊「複製 CSS」按鈕複製代碼
7. 將代碼套用到 Bloxd.io 遊戲客戶端

## 技術細節

- **前端框架** - 純 HTML/CSS/JavaScript
- **CSS 預處理** - 支援漸層、陰影、動畫等高級效果
- **配置文件** - 多個 CSS 檔案支援不同功能模組
- **響應式設計** - 支援桌面和行動設備
